# RPA_HWP

## 여러 한글 파일을 하나의 한글 파일로 합치기

- 작업폴더 1, 작업폴더 2 안에 각각 '제1회', '제2회', .. 와 같이 폴더가 들어있고 그 안에 A, B, C 한글파일이 들어있는 상황
- 목표 : 각 작업폴더에서 A끼리만 모아서 역순으로 하나의 한글 파일로 합치기
